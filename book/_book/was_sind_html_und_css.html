Welche der folgenden Aussagen sind korrekt ?

HTML (Hypertext Markup Language) ist für die Struktur der Webseite verantwortlich. CSS (Cascading Style Sheets) ist für die Datenübertragung zum Webserver verantwortlich. HTML (Hypertext Markup Language) ist für die Hosting-Funktionen verantworlich. CSS (Cascading Style Sheets) ist für die Formatierung und für das Layout der Webseite verantwortlich.
